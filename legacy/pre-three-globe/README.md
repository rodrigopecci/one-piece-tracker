# Pre-Three.js globe backup

This directory contains the production globe implementation that was replaced by
the approved Three.js version on 2026-07-21.

Backed-up files:

- `index.html`
- `app.js`
- `globe.html`

To restore it, copy these three files back to the project root. The backup is
kept separate so the current application can evolve without losing the previous
globe implementation.
